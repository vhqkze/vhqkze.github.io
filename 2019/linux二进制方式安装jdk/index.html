下载安装jdk

去oracle下载jdk ，然后解压，将解压内容移动到合适的目录

1

$ sudo mv jdk1.8.0_202 /usr/lib/java

配置

编辑/etc/profile文件

1

$ sudo vim /etc/profile

在文件末尾增加以下内容

1
2

export JAVA_HOME=/usr/lib/java/jdk1.8.0_202
export PATH=$PATH:$JAVA_HOME/bin

保存后使配置生效

1

$ source /etc/profile
