解决ls命令中文文件名显示一串问号的问题
发表于 2018-05-20 | 更新于 2018-05-20
Linux
Linux

使用ssh连接树莓派后，使用ls命令，其中中文文件名显示为一串问号，使用env命令查看环境变量，发现LANG=en_GB.UTF-8、LC_CTYPE=zh_CN.UTF-8，尝试将变量LC_CTYPE的值由zh_CN.UTF-8改成en_GB.UTF-8后，中文文件名显示正常。
